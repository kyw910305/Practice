Table Header

Use the TH element to define table headers.

Fistname Lastname Age
Jill Smith 50
Eve JackSon 94










Vertical Table Headers

The first column becomes table headers if you set the first table cell in each table row to a TH element:

FirstName Jill Eve
Lastname Smith Jackson
Age 50 94










Left-align Headers

To left-align the table headers, use the CSS text-align property.

Name Age
Jill Smith 50
Eve JackSon 94










A header that spans two columns

User the colspan attribute to have a header span over multiple columns.

Fistname Lastname Age
Jill Smith 50
Eve JackSon 94










Table Caption

To add a caption to a table, use the caption tag.

Monthly savings
Month Savings
January $100
February $50
